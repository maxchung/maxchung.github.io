et vos lieux enregistrés.

Appuyez sur le bouton pour voir votre place sur une carte.

Le téléphone affiche à nouveau la page principale.
Il y a un bouton d'aide.
Vous appuyez sur le bouton d'aide.

ambulance.
Choisissez le type d'aide dont vous avez besoin.
Vous appuyez sur une ambulance.
On vous demande si vous voulez appeler le 911.
Vous avez le choix entre oui et non.
Appuyez sur le titre pour entendre le message.
Vous appuyez sur « Oui ».
